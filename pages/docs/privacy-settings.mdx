## Team Roles

The site owner has Administrator privileges, and they may assign other administrators.

### Role: Administrator

In addition to all "manager" privileges, Site Administrators may:

- Inviting or revoking Team members and setting their role (Admins may invite other administrators)
- Setting the site to be public or private
- Creating API tokens

### Role: Manager

In addition to all "Writer" privileges, Site Managers may:

- Modify the Schema

### Role: Writer

In addition to all "Reader" privileges, Site Writers may:

- Add record nodes to a Record Set
- Patch or rewrite the value of Site Records

### Role: Reader

Site Readers may:

- List all nodes
- Read all nodes
- Read the schema for all nodes

When the site is set to "public", anyone may be a Reader of the Site.

## API Access

You may add API access that allows programatic read+write access to your data.
