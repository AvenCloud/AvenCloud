- Create a site
- Create Site API Token
- Make site public to GET anonymously

## HTTP API

Get Site Schema:

```
GET https://aven.io/api/v1/{SITE_NAME}/_schema
```

List Site nodes:

```
GET https://aven.io/api/v1/{SITE_NAME}/_children
```

For a record node:

```
GET https://aven.io/api/v1/{SITE_NAME}/{NODE_ADDRESS}
```

```
PUT https://aven.io/api/v1/{SITE_NAME}/{NODE_ADDRESS}
```

For a record set node:

```
GET https://aven.io/api/v1/{SITE_NAME}/{NODE_ADDRESS}/_children
```

```
POST https://aven.io/api/v1/{SITE_NAME}/{NODE_ADDRESS}
```

For records in a record set:

```
GET https://aven.io/api/v1/{SITE_NAME}/{NODE_ADDRESS}
```

```
PUT https://aven.io/api/v1/{SITE_NAME}/NODE_ADDRESS
```

```
DELETE https://aven.io/api/v1/{SITE_NAME}/NODE_ADDRESS
```

## WebSocket API
