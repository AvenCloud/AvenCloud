Aven Cloud is a web development ecosystem built on top of popular and robust tools.


### Internals

Aven Cloud is built on:

- GitHub
  - GH Actions
- VSCode
- DigitalOcean VPS
  - Debian
  - Systemd services
- Docker
  - Compose
- PostgreSQL
- Prisma
- JWT
- Next.js
  - Webpack
  - React
  - next-mdx-enhanced
- Emotion


### Development

Our upcoming electron app will help you create and run a web application on your PC or Mac.

### Self-Hosted Deployment

Our upcoming tools will help you setup and maintain your site on Digital Ocean, which will cost as little as $5/month (plus the cost of your domain).

### Hosted

Eventually Aven will offer a hosted version of Aven Cloud, so you won't worry about server management or uptime.
