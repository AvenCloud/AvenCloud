This is Sky, the first prototype Aven community, hosted at [aven.io].

Fully open source at: https://github.com/avencloud/sky

## Run locally

To run Aven Sky on your computer, make sure you have Node.js 12 or 14, and Yarn.

To run this site, check out the repo: https://github.com/avencloud/sky

```
git clone git@github.com:AvenCloud/sky.git
cd sky
yarn
yarn start
```
