# Values (under consideration)

- consistency and focus
- diversity and inclusion
- build incrementally
- eliminate complexity
- expose underlying tradeoffs
- make it beautiful
- open culture
- stand on stable

# How to pick technology

- Open source
- Long history, if possible
- Community + commercial support
- Prioritize Simplicity
- Don't waste versatility

# How to pick services

- Use services with low lock-in
- Easiest and most affordable
