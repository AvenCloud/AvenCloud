

So you want to set up your own community with the Aven platform. Let's do it!

This process will be a bit more technically involved than Aven [Pro](/pro) or [Enterprise](/enterprise).

When you go this route, the fate of your website and community is in your hands! Hopefully the content on this site has helped prepare you.

***

- Acquire a domain Name
- Set up your DNS on CloudFlare
- Set up your development computer
- Set up your production VPS on DigitalOcean
- Set up your database
- Add integrations for email, video, and payments
- Set up the Aven production server
- Configure continuous deployment from your git server
- Modify your site's content and theme
