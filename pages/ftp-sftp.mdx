
What if you want to edit files on another computer? It would be great if there was a standard way to do that!

## FTP

A protocol for sharing files and folders over TCP/IP, typically on port 21.

## SFTP

Known as SSH File transfer protocol, this is the most convenient secure FTP option, because it is built in to SSH.

## FTPS, FTP over SSL, oh my!

There are other options for secure FTP, [I've found this article to be most helpful](https://blog.devolutions.net/2016/05/ftps-sftp-or-ftp-over-ssh).
