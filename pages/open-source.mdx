"Open Source" is the name for software that with public source code and a license that allows anybody to freely see and modify the source code. With many open source projects, you are even allowed to modify the software and resell it as your own product.

### Licenses

See the [Open Source Licensing](/open-source-licensing) page to learn about different types of Open Source Software.

### Aven Open Source

Aven follows an [open philosophy](/open-philosophy). The source code for this site is licensed under the [Apache 2.0](https://github.com/AvenCloud/sky/blob/main/LICENSE.md) license.
