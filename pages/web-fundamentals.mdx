# Web Fundamentals

In this course, we discuss the technology that powers the web, and we will build a custom website from scratch using entirely open-source technology. In this course, you will learn:

---
- Set up a Linux web server
- Set up your domain name
- Host a static website
- Secure your site with HTTPS and other technology
- Become familiar with HTML, CSS, and React
- Set up a next.js app on your server
- Write content for your website in markdown
- Host your code on Github
- Continuously test and deploy code when it changes
- Optimize your site for social media and search engines
- Accessibility
---
