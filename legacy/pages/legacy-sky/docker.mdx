## Docker Containers

An isolated container that will run the same on every computer. Often you have one docker container per application.

## Docker Hub

The registry to publish you docker containers

## Docker Compose

A utility to run several pre-configured containers on a computer together.

## Kubernetes

A complicated system (involving other tools such as Helm), that allows you to coordinate docker containers on a fleet of computers.
