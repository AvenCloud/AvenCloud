Smart developers value their freedom, and care about the longevity of their stack.

**[Aven is entirely Open Source](https://github.com/AvenCloud/cloud), under the liberal [Apache 2.0 license](http://www.apache.org/licenses/LICENSE-2.0).**

Aven Cloud builds upon several well-loved and well-maintained open source projects:

- TypeScript
- NextJS
- React
- Chakra UI
- Webpack
- NodeJS
- Docker
- Prisma
- Postgres
- AJV
- [and several others](https://github.com/AvenCloud/cloud/blob/main/package.json)
